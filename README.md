# 🧠 Stroke Prediction Application

## 📌 Overview
This project is a machine learning-based application that predicts the risk of stroke using patient health data. It helps in early detection and supports decision-making through a simple graphical interface.

## 🚀 Features
- Predict stroke risk using trained machine learning models
- User-friendly GUI for easy interaction
- Real-time prediction results
- Data preprocessing and feature engineering

## 🛠️ Technologies Used
- Python
- Scikit-learn
- Pandas, NumPy
- Tkinter (GUI)

## 📊 Model Details
- Logistic Regression
- Random Forest
- Model Accuracy: **91.89%**
